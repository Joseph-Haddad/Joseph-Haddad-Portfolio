Microcontroller
Fire Alarm

This project features a solution to a fire alarm system. It has been developed in c using the arduino microcontroller on tinkercad. (As pictured Below)

The purpose of this project is to develop a microcontroller based solution to something. In this case I have selected a fire alarm. This solution had to be designed to take full advantage of the Arduino and all the features it comes with. These Specifications are as shown below.

Criteria

  • Digital I/O – switch
  • Digital I/O – Interrupt based debouncing
  • Digital I/O – LED
  • Analog Input
  • Analog Output – PWM
  • Serial I/O
  • LCD
  • Timers

The design provides a fire alarm system. Whilst looking to address an indoor environment improvement system. That is able to monitor and protect an environment such as your home or office so that both the users and the building can be safe.

Adhering to these criteria created some challenges and learning oppotrunities. Due to COVID-19 restrictions. Designing this physically was not an option, therefore development on tinkercad was conducted. This proved to be a further challenge, as simulation does not run as fast as real life. Which made it difficult to validate the timers and input compare registers.

Something I would like to explore in future is revdeveloping the design using physical components. Seeing if it would function similarily. As simulation is not entirely the same as real life. Below is a video of the solution in action on tinkercad.
